# Política AI-safe

Podés usar IA como apoyo, pero tu portafolio debe mostrar aprendizaje verificable. Cada entrada debe declarar el uso de IA, explicar decisiones propias, ser reproducible y poder defenderse sin asistencia.

## Niveles A0-A4

| Nivel | Uso permitido | Evidencia requerida |
|---|---|---|
| A0 | Sin IA durante el componente evaluado. | Checkpoint, defensa o variación en vivo. |
| A1 | IA como tutora. | Pregunta, respuesta y decisión tomada. |
| A2 | IA como copiloto. | Diff revisado, pruebas y explicación. |
| A3 | IA como co-diseñadora. | Alternativas, riesgos y decisión final. |
| A4 | IA con herramientas. | Logs, permisos y aprobación humana. |

No publiques claves, tokens, datos privados ni conversaciones completas con información sensible.
